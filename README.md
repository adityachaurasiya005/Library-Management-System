# 📚 Library Management System

A simple command-line Library Management System built using Python and file handling.

## Features

- ➕ Add Book
- 📖 View Books
- 🔍 Search Book
- ✏️ Update Book
- ❌ Delete Book
- 📚 Borrow Book
- ↩️ Return Book

## Technologies Used

- Python
- File Handling
- VS Code
- Git & GitHub

## Project Structure

```
Library-Management-System/
│
├── library.py
├── books.txt
├── README.md
└── .gitignore
```

## How to Run

1. Clone the repository

```
git clone https://github.com/adityachaurasiya005/Library-Management-System.git
```

2. Open the project folder

3. Run

```
python library.py
```
